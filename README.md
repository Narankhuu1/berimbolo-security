# Berimbolo Security 🛡️

**Аюулгүй байдлын тэргүүлэгч систем** — Монголын аюулгүй байдлын компанийн вэб сайт бүтээн байгуулалт.

## 📁 Файлын бүтэц

```
berimbolo-security/
├── index.html          # Үндсэн HTML бүтэц
├── styles.css          # Бүх загвар, анимац
├── script.js           # Бүх JavaScript функц
└── README.md           # Энэ файл
```

## 🚀 Үндсэн боломжууд

- **Уян хатан Landing хуудас** — Hero хэсэг, үйлчилгээ, үнийн багц, сэтгэгдэл, холбоо барих
- **Үйлчилгээ захиалах** — Аюулгүй байдлын систем суурилуулах захиалга
- **Live Chat Widget** — Хэрэглэгчийн тусламжийн чатбот
- **Ажилтны самбар (Dashboard)** — Бүрэн ажилтны портал:
  - Ерөнхий тойм / Аналитик
  - Цаг товлолтын хуваарь
  - Хэрэглэгчийн удирдлага
  - CCTV хяналтын дүрс
  - Инженерийн ажлын даалгавар
  - Ажилтнуудын мэдээлэл
  - Тайлан, экспорт
- **Нэвтрэлт** — Эрхэд суурилсан нэвтрэлт (Удирдлага, Оффис, Инженер)
- **Local Storage** — Захиалгыг хөтөчид хадгалах

## 👥 Туршилтын нэвтрэх мэдээлэл

## 🛠️ Ашигласан технологи

- HTML5
- CSS3 (Custom properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (Фреймворкгүй)
- LocalStorage — мэдээлэл хадгалах

## 📦 Хэрхэн ашиглах

1. Репо-г clone хийх
2. `index.html` файлыг аль ч орчин үеийн хөтөчид нээх
3. Сервер эсвэл build алхам шаардлагагүй

## 📝 Багш / Удирдагчид зориулсан тэмдэглэл

Энэ бүтээн байгуулалт дараах чадварыг харуулж байна:
- Орчин үеийн CSS архитектур (custom properties)
- JavaScript төлөвийн удирдлага, DOM манипуляц
- Modal систем, маягт боловсруулалт
- Бодит цагийн самбар (data table-уудтай)
- Эрхэд суурилсан нэвтрэлтийн урсгал
- Responsive, mobile-first дизайн
